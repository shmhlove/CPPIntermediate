-> this*가 파라미터 첫번째값으로 들어간다.(어셈블리코드로 확인해보면 this*가 실제로 있다)
--------------------------------------------------
일반 함수포인터로 구성된 파라미터로 멤버함수를 넘기기 위해서는
static 함수를 선언해야하며
static 함수 내에서 멤버에 접근하기 위해서는 this가 필요한데
파라미터로 this를 받을 수 있다면 그렇게 하고,
그럴 수 없다면 thisMap을 이용해서 처리한다.
--------------------------------------------------
다중상속에 있어서 this문제
class C : public A, public B 일때
C c;
A * a = &c;
B * b = &c;

a와 b의 value가 다르다.
b의 value 는 c + sizeof(A)의 value가 들어간다.
?? 멤버에 동적객체가 있을때는 sizeof가 어떻게 되는건가? 단순히 포인터만큼만 늘어나겠는데?? 동적 데이터는 힙영역에 잡힐테니.
